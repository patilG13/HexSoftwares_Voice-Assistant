Quick Setup
Create virtual environment:

bash
python -m venv venv

# Activate:
# Windows: venv\Scripts\activate
# Mac/Linux: source venv/bin/activate
Install packages:

bash
pip install -r requirements.txt
Run:

bash
python Jarvis.py
Basic Commands
time / date - Current time/date

open google - Open websites

search cats - Web search

note buy milk - Save note

timer 5 - Set 5-minute timer

system - System info

joke - Tell a joke

help - Show all commands

exit - Quit

Features
Voice & text input

System monitoring (CPU, RAM, battery)

Open websites

Notes & reminders

Calculator & timer

Screenshots

Customizable name & voice

Need Help?
Check microphone settings

Make sure venv is activated

Run as admin if mic not detected

Say "voice mode" or "text mode" to switch input methods!
